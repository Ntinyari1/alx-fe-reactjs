# Tailwind CSS + React Integration

A foundational project demonstrating the setup and configuration of **Tailwind CSS v4** within a **React** application powered by **Vite**. This project serves as a starting point for applying utility-first styling principles to modern web components.

##  Features

- **Vite 7**: Ultra-fast frontend tooling and bundling.
- **React 19**: The latest React features for building user interfaces.
- **Tailwind CSS v4**: Seamless integration using the `@tailwindcss/vite` plugin for lightning-fast CSS processing.
- **Responsive Design**: Mobile-first approach using Tailwind's utility classes.
